Projects

Addressing Data Heterogeneity in Federated Learning

Investigated challenges of non-IID data in Federated Learning (FL). Developed novel latent space alignment techniques such as Gradient Harmonization Between Losses (FedAlign) and Adversarial Feature Alignment to mitigate client drift. Designed an experimental workflow integrating alignment and classification losses to evaluate performance across diverse FL scenarios, achieving state-of-the-art accuracy of 76.88% on CIFAR-10 dataset with label and quantity skew, outperforming baseline FedAvg and other benchmark techniques.

Landmark 3D Reconstruction and App Deployment

Developed an Augmented Reality (AR) Android app to visualize a 3D reconstruction of the Pantheon using computer vision techniques. Implemented feature detection (SIFT), robust feature matching (FLANN-based KNN), and Structure from Motion (SfM) techniques to process 1401 image pairs. Utilized OpenCV for preprocessing workflows such as histogram equalization and RANSAC-based outlier removal to improve reconstruction accuracy and camera pose estimation.

Model Compression Techniques for Deep Learning Models

Implemented and evaluated pruning, quantization, and knowledge distillation on VGG architectures with CIFAR-100 to enhance efficiency for resource-limited environments. Utilized structured and unstructured pruning methods based on L2 norms for channel removal. Compared Post-Training Quantization (PTQ) and Quantization-Aware Training (QAT) across different bit-widths (int8, int4, float16, bfloat16), demonstrating significant accuracy retention with QAT. Used logit matching, hint-based distillation, and contrastive representation distillation (CRD) to closely approximate teacher model performance.

Course Recommendation System Using Retrieval-Augmented Generation (RAG)

Developed a course and instructor recommendation system using a Retrieval-Augmented Generation (RAG) pipeline with LangChain, designed to assist students in making informed course selections. Scraped and preprocessed course and instructor reviews from multiple sources, addressing challenges like unofficial course nicknames and abbreviations. Integrated Mistral-7B-Instruct LLM with ChromaDB vector database for efficient information retrieval. Deployed the system using Hugging Face Spaces and Gradio, making it accessible for personalized recommendations.

Succession Planning Software Project

Developed an ML-driven HR management platform for succession planning and workforce optimization in collaboration with Devsinc. Created regression-based predictive models using historical performance data to assess employee KPIs, identify high-potential candidates, and highlight at-risk employees. Implemented security features like role-based access control, two-factor authentication, and data encryption to safeguard sensitive employee data. Integrated mentor-mentee matching, personalized training recommendations, and progress tracking features based on skill gaps and career goals to enhance employee development.
